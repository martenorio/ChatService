
# Proyecto de Chat en Tiempo Real con SignalR y React

Este es un proyecto de demostración que implementa un chat en tiempo real utilizando SignalR en el backend con .NET Core 8 y React en el frontend. La aplicación permite a los usuarios conectarse y comunicarse en salas de chat en tiempo real.

## Funcionalidades

- Creación y unión a salas de chat.
- Mensajería en tiempo real dentro de las salas.

## Requisitos previos

- .NET Core 8 SDK
- Node.js y npm (para el frontend)
